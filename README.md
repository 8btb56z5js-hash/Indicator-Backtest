# DCS v4.6 Elite Indicator

Repository for the elite TradingView indicator project.

## Structure
- `engine/` – Python modules (refactored from Pine)
- `metadata/` – Instrument metadata CSVs
- `param_adapter/` – Dynamic parameter functions
- `backtest/` – Lean back‑test script & results
- `model/` – Qlib confidence model
- `pipeline/` – TradingAgents CLI pipeline
- `demo/` – Lightweight‑Charts demo page
- `final/` – Final Pine indicator file
